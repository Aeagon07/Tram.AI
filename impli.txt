6️⃣ Animation System

Use one animation config file:

  hidden: { opacity: 0, y: 20 },
  visible: { opacity: 1, y: 0 }
}

Use consistent duration: 0.3–0.5s

Avoid over-animation.

Premium apps are subtle.
